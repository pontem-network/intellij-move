# Move on Endless language plugin for the IntelliJ-based IDEs

The plugin supports the latest version of the Move on Endless language, including the new features of v2, like enums or receiver-style functions.
Plugin documentation is available at https://intellij-move.github.io/.

## Installation

Open `Settings > Plugins > Marketplace` in your IDE, search for _Move on Endless_ and install the plugin.

## Compatible IDEs

The plugin is compatible with all IntelliJ-based IDEs starting from 2024.2.
On the other versions, older versions of the plugin are available.

New features are announced in [What's New section of the documentation](https://intellij-move.github.io/what-s-new.html).

## Features

* [Move on Endless](https://endless.labs) language support, including Compiler-V2 features.
* Rename refactoring.
* Code formatter.
* Extensive integration with `endless-cli`.
* Running Move on Endless compiler on-the-fly as you type.
* Integration with the Endless Decompiler.
